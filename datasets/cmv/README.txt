1.Data Size: The whole dataset consists of 11,565 training instances and 1,481 testing instances.

2. Data Format: The composition of each line is as follows. 
 
q_context # q # p_r # p_r_context # n_r_1 # n_r_1_context # n_r_2 # n_r_2_context # n_r_3 # n_r_3_context # n_r_4 # n_r_4_context

Note:  
(1)q: quotation
(2)q_context: the context of quotation
(3)p_r: positive reply
(4)p_r_context: the context of positive reply
(5)n_r_i: i-th negative reply
(6)n_r_i_context: the context of i-th negative reply
